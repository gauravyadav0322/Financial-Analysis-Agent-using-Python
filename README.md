# Financial Analysis Agent using Python

## About This Project
This project is part of my learning journey in Data Science and AI, where I explored how to build a simple financial analysis system using Python.

The goal of this project is to process and analyze financial data from company reports and extract useful insights. The project uses real-world financial data from company filings to understand financial performance and key metrics.

---

## Objective
The main objective of this project is to:

- Understand financial statements
- Extract and analyze financial data
- Build a basic analysis workflow using Python
- Learn how AI/automation can assist in financial decision-making

---

## Dataset
The project uses real financial report data from Amazon's quarterly filing (Form 10-Q).

This includes:
- Income statement
- Balance sheet
- Cash flow statement

## Project Workflow
1. Load financial data from the report  
2. Extract important financial metrics  
3. Analyze trends such as:
   - Revenue growth  
   - Profitability  
   - Cash flow changes  
4. Present insights in a structured format  

---

## Example Insights
From the financial data:

- Total net sales increased from 143,083 million to 158,877 million in quarterly comparison  
- Net income showed significant growth, indicating improved profitability  
- Operating cash flow also increased, reflecting stronger business operations  

These insights are based on financial statements provided in the report.
